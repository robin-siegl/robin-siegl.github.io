Slotmachine

User Interface

Here is a short description about the user interface and how you can interact with the game.

Start a new game or load a previous game:

After you start a new game, you will get some Money and start at Level 0.

You can load a previous game "Load Game", if there is one. It will load a new game if there is no saved game.

Statusbar - Money:

Here you can see your current amount of money. It automatically updates if you lose or win money.

Statusbar - Level:

If you gain XP in-game, this XP-Bar updates. After you gained enough XP you will level up. You can see your current level in the red circle.

Slotdisplay:

Here you can see the random symbols appear. If you have the same symbols, they will get highlighted.

Coins per Game:

In-game you can choose the amount of coins you want to pay per game. You can decrease with the "-" button or increase with the "+" button.

The current amount will be displayed between "-" and "+" button.

If you reache the minimum amount of coins per game (1$), the "-" will be disabled.

If you reache the maximum amount of coins per game (64$), the "+" will be disabled.

Difficulty:

You can increase the difficulty by clicking on the button until it reaches 3, after this it will jump back to 1.

Insert Coin:

Here you start the process where the coins per game will be removed from your money and get a random symbole on every slotdisplay.

The button will be disabled if you don't have enough money. You can decrease the coins per games to enable it again.

Game Messages:

Everything that is happening in the game is shown here.

This is a example about the game messages. Here you can see the amount of money you won.

Save current game or load a previous game:

On the bottom of the gamescreen you will find a save game button, to save the current game local.

You can also load a previous game. It will load a new game if there is no saved game.

Symbols

XP and Level

You will gain +5 (on difficulty 1), +10 (on difficulty 2) and +15 XP (on difficulty 3) per game. After you won you will gain +10 (with 2 symbols), +20 (with 3 symbols), +30 (with 4 symbols), +50 (with 5 symbols) and +75 XP (with 6 symbols).

If your level gets higher, it will be harder to level up.
